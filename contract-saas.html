🚀 실제 서버 연동 가이드
📌 현재 상태
이 파일은 브라우저 localStorage에 데이터를 저장합니다. 실제 서비스로 전환하려면 아래 3단계를 따르세요.
1
Supabase 프로젝트 생성 (무료)
# 1. supabase.com 가입 후 새 프로젝트 생성
npm install @supabase/supabase-js

# 2. .env 파일에 추가
SUPABASE_URL=https://xxxx.supabase.co
SUPABASE_ANON_KEY=your-anon-key
2
DB 함수 교체 (localStorage → Supabase)
// 현재 코드 (localStorage)
const events = DB.get('events');

// ↓ Supabase로 교체
const { data: events } = await supabase
  .from('events')
  .select('*')
  .order('created_at', { ascending: false });
3
카카오톡 공유 SDK 연동
<!-- HTML head에 추가 -->
<script src="https://t1.kakaocdn.net/kakao_js_sdk/2.7.2/kakao.min.js"></script>

// JavaScript에서 초기화
Kakao.init('YOUR_KAKAO_APP_KEY');

// developers.kakao.com 에서 앱 등록 필요
// 플랫폼 → 웹 → 사이트 도메인 등록
⚠️ 카카오 앱키는 developers.kakao.com 에서 발급받고,
JavaScript 키를 사용하세요. 반드시 도메인 등록이 필요합니다.
4
Vercel 배포 (무료)
# GitHub에 push → Vercel 자동 배포
npm i -g vercel
vercel login
vercel --prod

# 무료 도메인: yourapp.vercel.app
